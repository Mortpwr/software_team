UX Strategy

高校内部党团学 App 高保真可交互原型

围绕“查看资讯 -> 报名志愿活动 -> 学习打卡 -> 个人积分沉淀”的成长闭环，将通知触达、活动转化、学习督促和个人画像沉淀整合为一个轻量、年轻、可扩展的校园移动体验。

学生痛点

资讯分散、报名入口难找、学习任务容易遗忘、个人荣誉和积分沉淀不可见。

管理员诉求

精准触达、活动容量管理、学习完成率统计、可追溯的学生成长数据。

Color

科技蓝、活力橙、青绿色和柔和玫瑰色，兼容深色模式扩展。

Layout

8pt 网格、大圆角卡片、无边框留白与弥散阴影。

Glass

顶部导航和底部 Tab Bar 使用毛玻璃，模拟 iOS 原生层次。

Motion

Fade-up 入场、ripple、弹性 Tab、骨架屏和阶梯滚动动画。
