# 高校内部党团学 App 高保真原型

## UX 与信息架构

### 用户画像

- 在校大学生：希望快速看到与自己相关的资讯、活动和学习任务，不想在多个群聊和系统之间来回确认；核心诉求是少错过、易报名、能打卡、成长记录可见。
- 校团委 / 学院管理员：需要统一发布资讯、组织志愿活动、跟踪青年大学习完成情况，并沉淀可统计的学生成长数据；核心诉求是触达精准、报名可控、过程可追踪、结果可复盘。

### 核心闭环

1. 查看资讯：首页聚合团日活动、志愿服务、校园文化等信息。
2. 报名活动：学生进入活动页筛选志愿 / 实践活动，完成报名并获得签到入口。
3. 学习打卡：学习页展示青年大学习、团章团史和安全教育任务，支持进度反馈。
4. 积分沉淀：个人中心沉淀志愿时长、学习积分、报名记录和成长轨迹。

### Tab Bar 结构

- 资讯：首页资讯、重点通知、快捷入口、近期动态。
- 活动：志愿活动筛选、容量进度、报名反馈、积分激励。
- 学习：青年大学习、学习进度、连续打卡、本周节奏。
- 我的：个人身份、先锋指数、积分资产、成长轨迹、证书与档案入口。

## Design System

- 色彩：科技蓝 `#2563eb`、活力橙 `#f97316`、清透青 `#06b6d4`、柔和玫瑰 `#fb7185`，通过渐变和低饱和背景降低传统党团系统的沉闷感。
- 布局：遵循 8pt 网格；卡片采用 `2xl/3xl` 大圆角、无边框视觉和柔和弥散阴影。
- 毛玻璃：顶部导航和底部 Tab Bar 使用 `backdrop-filter: blur(24px)`，形成接近 iOS 的层级感。
- 动效：页面 `fade-up`，按钮 `scale + ripple`，Tab 弹性激活，卡片 hover 上浮，骨架屏 shimmer，列表进入视口 staggered reveal。
- 扩展性：样式变量放在 `assets/shared.css`，页面数据与交互工具放在 `assets/shared.js`，可迁移到正式 Vue 组件体系。

## 文件结构

```text
web/prototype/
  index.html          # iPhone 15 Pro 展示画廊
  home.html           # 首页资讯
  activity.html       # 活动报名
  study.html          # 学习打卡
  profile.html        # 个人中心
  assets/shared.css   # 共享视觉规范与动画
  assets/shared.js    # 共享模拟数据与交互工具
```

## 预览方式

直接打开 `web/prototype/index.html`，或在 `web/` 目录运行现有 Vite 服务后访问：

```bash
npm run dev -- --host 0.0.0.0
```

然后访问 `/prototype/index.html`。
