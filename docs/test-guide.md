# 平台使用说明

本文档面向使用本平台进行体验、测试和互测的同学，主要说明登录方式、页面功能和使用边界。

## 1. 访问入口

- 平台地址：`http://10.10.0.21/`
- 推荐浏览器：Chrome、Edge
- 当前主交付端为 `web/ + backend/`
- 根目录微信小程序代码仅作过程材料保留，不作为当前正式使用入口

## 2. 登录方式

系统使用“学号 + 密码”登录，不需要手动选择角色。

默认初始密码规则：

```text
Stu@ + 学号后 6 位
```

常用测试账号如下：

| 角色 | 学号 | 默认密码 |
| --- | --- | --- |
| 学生 | `2024201581` | `Stu@201581` |
| 三级协同管理者 | `2023200444` | `Stu@200444` |
| 管理老师 | `2022200999` | `Stu@200999` |
| 学院领导 | `2024210888` | `Stu@210888` |

说明：

- 若账号密码已被改动，请联系项目组成员重置
- 管理老师可在工作台中重置学生密码
- 登录后系统会自动按账号返回对应角色界面

## 3. 页面说明

### 3.1 首页

- 展示通知公告、站内消息、个人待办和常用入口
- 不同角色登录后，首页展示内容会有所区别

### 3.2 知识库

- 支持政策、流程、常见问题等内容检索
- 支持查看详情、收藏、最近浏览、热门内容
- 部分条目带有官方链接或附件，可直接查看或下载

### 3.3 党团页

- 展示入党、入团相关阶段进度、历史记录和待办事项
- 提供官方依据、官方文档、校历等资料
- 提供理论学习与自测能力
- 支持上传部分流程材料

### 3.4 办事申请

- 支持常见办事场景的在线填写、保存和提交
- 支持草稿保存
- 支持查看申请状态、审批结果和历史记录
- 部分申请支持文档生成、下载或打印

### 3.5 通知与消息

- 可查看公告、站内消息及相关提醒
- 不同角色看到的内容范围不同

### 3.6 荣誉展示

- 展示学生或集体荣誉信息
- 支持按条件筛选
- 部分条目附带证明材料

### 3.7 学业分析

- 展示培养方案、学分完成情况和缺口分析
- 支持上传 PDF 成绩单进行识别
- 支持查看风险提示和修读建议

### 3.8 个人信息

- 可查看本人基础资料
- 可维护扩展信息
- 可修改个人密码

### 3.9 工作台

工作台主要面向管理老师、协同管理者和学院领导。

支持的功能包括：

- 查看统计概览
- 审批申请
- 发布通知和查看批次
- 管理知识库、荣誉、模板和部分材料
- 查看学生画像、学业风险和审计日志
- 查看党团进度与相关后台数据

## 4. 文件使用说明

- 平台部分功能支持文件上传、下载和预览
- 上传后，文件通常会附着在对应业务记录中
- 不同角色对文件的可见范围不同
- 若模板文件尚未上传，部分模板下载功能可能不可用

## 5. 使用边界

为避免误解，使用时请注意：

- 平台默认连接真实后端，不以浏览器本地 mock 作为正式使用链路
- 真实短信、微信 OAuth、电子签章、复杂选课等能力不在当前项目范围内
- 部分邮件、短信、外部通知能力属于课程场景下的简化实现
- 若某些页面数据较少，不一定表示页面异常，也可能是当前测试数据有限

## 6. 常见问题

### 6.1 登录失败

请先确认：

- 学号是否正确
- 密码是否符合默认规则
- 该账号密码是否已经被修改

### 6.2 页面能打开，但部分内容为空

常见原因：

- 当前账号没有对应权限
- 该模块当前数据较少
- 该内容依赖前一步操作结果

### 6.3 上传后无法预览或下载

- 可能是文件类型、大小或权限范围不符合当前模块要求
- 也可能是该业务对应的模板或附件尚未准备完整
